Termemum

Terminal emulator written in C for X11.

Termemum is a terminal emulator written in C for X11. I made it as a challenge for myself. The terminal emulator supports all unix commands since it was built with libVTerm. Termemum support basic customizing of the terminal through a C header file that works as a config (kinda like STs config). There is a downside of using a header file since the whole project needs to be recompiled after changing the config file.

It has been tested to work with Wayland on mulitple desktop environments and window managers. Even though it was built for X11.

The project is extremely lightweight but I'm planning to add more features in the future. Termemum is open source and can be found and installed on my github. Termemum
